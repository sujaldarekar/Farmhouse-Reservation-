### Prerequisites

- Node.js installed

### Getting Started

1. **Clone the repository**

   ```bash
   git clone <your-repo-url>
   cd farmhouse
   ```

2. **Backend Setup**

   ```bash
   cd backend
   npm install
   # Create a .env file based on .env.example
   npm run dev
   ```

3. **Frontend Setup**

   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```

4. **Seed Database (Optional)**
   ```bash
   cd ../backend
   node seed.js
   ```
